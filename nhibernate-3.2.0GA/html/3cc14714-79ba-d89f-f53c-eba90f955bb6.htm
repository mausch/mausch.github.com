ICollectionAliases Interface Properties See Also Send Feedback

The ICollectionAliases type exposes the following members.

Properties

  NameDescription
Suffix
Returns the suffix used to unique the column aliases for this particular alias set.
SuffixedElementAliases
Returns the suffixed result-set column-aliases for the columns making up the collection's elements.
SuffixedIdentifierAlias
Returns the suffixed result-set column-aliases for the column defining the collection's identifier (if any).
SuffixedIndexAliases
Returns the suffixed result-set column-aliases for the columns making up the collection's index (map or list).
SuffixedKeyAliases
Returns the suffixed result-set column-aliases for columns making up the key for this collection (i.e., its FK to its owner).
